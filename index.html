Шпаргалка

  1. git commit - фіксує стан поточних змін і додає до репозиторію
  2. git status - відображає стан робочого каталогу та розділу проіндексованих файлів
  3. git add . - додає зміни з робочого каталогу до розділу проіндексованих файлів
  4. git checkout (....) - дозволяє переміщатися між гілками
  5. git push - команда, яка передає у віддалений репозиторій зміни, зроблені у локальному репозиторії
  6. git pull - використовується для вилучення та завантаження вмісту з віддаленого репозиторію та негайного оновлення локального репозиторію цим вмістом
  7. git branch - дозволяє переглядати які є гілкі
  8. git branch (.....) - дозволяє створювати нови гілкі
  9. git commit -am "комент що добавляє чи зміни" - зберігає всі зміни в файлі
  10. git push origin DZ#3 -відправляє всі зміни в вказану гілку
